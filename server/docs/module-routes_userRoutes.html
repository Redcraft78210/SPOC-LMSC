Module: routes/userRoutes

Routes Express pour la gestion des utilisateurs Ce module définit toutes les routes HTTP pour les opérations sur les utilisateurs, incluant la gestion des profils, l'authentification 2FA, et les opérations administratives. Toutes les routes nécessitent une authentification via middleware.
